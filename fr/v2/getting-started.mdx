# Démarrage avec Salsa as a Service v2.0

<VersionBanner version="v2" status="current" />

Bienvenue sur **v2.0** ! Ce guide vous aide à configurer votre compte et à exploiter les nouvelles fonctionnalités IA, les capteurs IoT et les intégrations avancées.

---

## Prérequis

Avant de commencer, assurez-vous d’avoir :

- Une adresse e-mail professionnelle valide  
- L’accès aux systèmes de votre restaurant (POS, inventaire, etc.)  
- Une connaissance de base de vos opérations actuelles  
- Les droits administrateur pour configurer les intégrations  
- (Optionnel) Des capteurs IoT pour la surveillance qualité  

<Callout type="info">
v2.0 inclut un assistant de configuration intelligent qui analyse votre profil et recommande une configuration optimale.
</Callout>

---

# Assistant de configuration rapide

<Steps>
  <Step title="Créer votre compte">
    Inscrivez-vous avec votre e-mail professionnel. Les clients entreprise peuvent utiliser SSO (Google, Microsoft, Okta, SAML).
  </Step>

  <Step title="Configurer votre profil">
    L’assistant IA analyse votre type de restaurant et suggère les paramètres optimaux.
  </Step>

  <Step title="Connecter vos systèmes">
    v2.0 détecte les systèmes compatibles et propose une intégration en un clic.
  </Step>

  <Step title="Importer les données historiques">
    Importez vos données de ventes et production pour entraîner le moteur IA.
  </Step>

  <Step title="Activer les fonctionnalités IA">
    Activez la prévision de demande, la prédiction qualité et le réapprovisionnement automatique.
  </Step>
</Steps>

# Configuration du compte

## Étape 1 : Création de compte

1. Visitez la [page d'inscription](https://app.salsaasaservice.com/signup)  
2. Choisissez votre méthode d’authentification :
   - E-mail + 2FA  
   - SSO (Google, Microsoft, Okta)  
   - SAML 2.0 entreprise  
3. Vérifiez votre identité  
4. Finalisez la vérification entreprise  

<Callout type="tip">
v2.0 prend en charge l’authentification biométrique, les clés matérielles et la connexion sans mot de passe.
</Callout>

---

# Configuration des fonctionnalités IA

## Prévision de demande

<Steps>
  <Step title="Activer la fonctionnalité">
    Allez dans **Fonctionnalités IA → Prévision de demande** et activez-la.
  </Step>

  <Step title="Connecter les sources de données">
    Liez votre POS, système de réservation et sources externes si nécessaires.
  </Step>

  <Step title="Importer les données historiques">
    Importez au moins 90 jours de données pour une précision optimale.
  </Step>

  <Step title="Définir les paramètres">
    Configurez l’horizon (7, 14 ou 30 jours) et les seuils d’alerte.
  </Step>
</Steps>

<Callout type="info">
L’IA atteint 70–80% de précision avec 90 jours de données et peut dépasser 95% après apprentissage continu.
</Callout>
