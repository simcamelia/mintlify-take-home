# Salsa as a Service API v2.0

The v2.0 API adds GraphQL support, enhanced webhooks, new AI forecasting endpoints, and improved analytics capabilities.

---

## Base URLs

**REST (Production)**

```text
https://api.salsaasaservice.com/v2
```

**GraphQL**

```text
https://api.salsaasaservice.com/graphql
```

**REST (Sandbox)**

```text
https://api-sandbox.salsaasaservice.com/v2
```

---

## Authentication

All API requests require authentication using your API key.

Get your API key from the Developer Settings in your dashboard:

https://app.salsaasaservice.com/settings/api

### Authorization Header

```http
Authorization: Bearer YOUR_API_KEY
```

**cURL**

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.salsaasaservice.com/v2/ingredients
```

**Fetch**

```javascript
fetch("https://api.salsaasaservice.com/v2/ingredients", {
  headers: { Authorization: "Bearer YOUR_API_KEY" }
});
```

---

## Rate Limits

- **Free Plan**: 1,000 requests per hour  
- **Professional Plan**: 10,000 requests per hour  
- **Enterprise Plan**: 100,000 requests per hour  

Rate limit headers are included in all responses:

```http
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200
```

## Inventory Management {#inventory}

### Get All Ingredients

```http
GET /ingredients
```

**Response**

```json
{
  "data": [
    {
      "id": "ing_123",
      "name": "Roma Tomatoes",
      "category": "vegetables",
      "current_stock": 25.5,
      "unit": "lbs",
      "quality_score": 8.5
    }
  ],
  "pagination": {
    "page": 1,
    "per_page": 50,
    "total": 1
  }
}
```

### Update Ingredient Stock

```http
PUT /ingredients/{ingredient_id}/stock
```

```json
{
  "quantity": 30.0,
  "unit": "lbs",
  "notes": "New shipment received"
}
```

## Analytics & Reporting {#analytics}

### Get Production Analytics

```http
GET /analytics/production?period=30d
```

```json
{
  "period": "30d",
  "total_batches": 45,
  "average_quality_score": 8.7
}
```

## Error Handling

```json
{
  "error": {
    "code": "INVALID_REQUEST",
    "message": "The request body is missing required fields"
  }
}
```

Common error codes:

- 400 Bad Request  
- 401 Unauthorized  
- 403 Forbidden  
- 404 Not Found  
- 422 Validation error  
- 429 Too Many Requests  
- 500 Internal Server Error  
- 502 Bad Gateway  
- 503 Service Unavailable
