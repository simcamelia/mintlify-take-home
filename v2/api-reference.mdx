## Authentication

All API requests require authentication using your API key.

<Callout type="info">
Get your API key from the <a href="https://app.salsaasaservice.com/settings/api">Developer Settings</a> in your dashboard.
</Callout>

### Authorization Header

<CodeGroup>
  <CodeBlock language="http" title="Header">
{`Authorization: Bearer YOUR_API_KEY`}
  </CodeBlock>

  <CodeBlock language="bash" title="cURL Example">
{`curl -H "Authorization: Bearer YOUR_API_KEY" \\
  https://api.salsaasaservice.com/v2/ingredients`}
  </CodeBlock>

  <CodeBlock language="javascript" title="Fetch Example">
{`fetch("https://api.salsaasaservice.com/v2/ingredients", {
  headers: { Authorization: "Bearer YOUR_API_KEY" }
});`}
  </CodeBlock>
</CodeGroup>

## Rate Limits

- **Free Plan**: 1,000 requests per hour  
- **Professional Plan**: 10,000 requests per hour  
- **Enterprise Plan**: 100,000 requests per hour  

<Callout type="warning">
Monitor the rate limit headers in responses to avoid throttling.
</Callout>

<CodeBlock language="http" title="Rate Limit Headers">
{`X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200`}
</CodeBlock>

## Inventory Management {#inventory}

### GET /ingredients

Retrieve all ingredients.

<Tabs>
  <Tab title="Request">
    <CodeBlock language="http">
{`GET /ingredients`}
    </CodeBlock>
  </Tab>

  <Tab title="Response">
    <CodeBlock language="json">
{`{
  "data": [
    {
      "id": "ing_123",
      "name": "Roma Tomatoes",
      "category": "vegetables",
      "current_stock": 25.5,
      "unit": "lbs",
      "quality_score": 8.5
    }
  ],
  "pagination": {
    "page": 1,
    "per_page": 50,
    "total": 1
  }
}`}
    </CodeBlock>
  </Tab>
</Tabs>

## Error Handling

### Error Response Format

<CodeBlock language="json">
{`{
  "error": {
    "code": "INVALID_REQUEST",
    "message": "The request body is missing required fields"
  }
}`}
</CodeBlock>

### Common Error Codes

<Accordion title="Client Errors">
- `400` Bad Request  
- `401` Unauthorized  
- `403` Forbidden  
- `404` Not Found  
- `422` Validation error  
- `429` Rate limit exceeded  
</Accordion>

<Accordion title="Server Errors">
- `500` Internal Server Error  
- `502` Bad Gateway  
- `503` Service Unavailable  
</Accordion>
